git objects and names

  • HEAD - reference to latest commit on current branch
  • FETCH_HEAD - reference to latest commit on fetched remote branch
  • .git - git repository directory
  • .gitignore - special file holding files' names and directories to be ignored.

git repository

  • git init - initializes an empty git repository
  • git status - displays status of repository
  • git log - displays log for current branch
  • git blame filename - displays file filename with annotated lines of last modifier
  • git remote add remote_name remote_address - adds new remote repository remote_name with address remote_address
  • git remote remove remote_name - removes remote remote_name
  • git remote - lists all remotes
  • git diff - show changes in repository

git index

  • git add . - add all files in current directory to index/stagin area
  • git checkout -- font.py - abandon changes in file font.py

git commit

  • git commit -a -m "message" - commits all files tracked by git and sets commit's message
  • git commit --amend - amends the previous commit with all tracked files
  • git show COMMIT - show changes introduced by COMMIT
  • git revert COMMIT -creates a commit that undoes changes introduced by COMMIT

git branch

  • git branch branch_name - creates a new branch named branch_name
  • git branch --set-upstream-to=origin/branch_name - sets current branch to track origin/branch_name
  • git checkout branch_name - switches to branch branch_new
  • git branch -d branch_name - drops branch branch_name

git push

  • git push - performs a push according to default push strategy
  • git push origin branch_name - pushes current branch to origin/branch_name
  • git push -f - forces(!) push to remote repository, effectively changing remote history
  • git push -u origin master - pushes to origin/master and sets current branch to track origin/master

git pull

  • git pull - performs a pull according to default pull strategy
  • git pull remote_name - performs a pull from remote_name

git fetch

  • git fetch - performs a fetch from default remote (origin)
  • git fetch remote_name - performs a fetch from remote_name

git merge

  • git merge branch_name - performs a merge of branch_name into the current branch
  • git merge -s 'strategy' ... - performs a merge of branch_name using the given strategy

git rebase

  • git rebase branch_name - performs a rebase of current branch on top of branch_name
  • git rebase -i HEAD~3 - performs a rebase on top of current branch in interactive mode. Can be used for squashing

git patch

  • git format-patch -1 HEAD - creates a new patch from last commit

git cherry-pick

  • git cherry-pick COMMIT_REF - performs a cherry pick of COMMIT_REF to current branch

git stash

  • git stash [--include-untracked] - performs a stash operation [includes untracked files]
  • git stash apply - applies latest stash
  • git stash list - list stashed changes
  • git stash drop - drops latest stashed change
